Proyectos

Calculadora

El proyecto consistió en desarrollar una calculadora básica utilizando un lenguaje de programación como Python. La calculadora fue diseñada para realizar operaciones aritméticas simples, como suma, resta, multiplicación y división. Se implementaron funciones para manejar cada una de estas operaciones, así como para manejar errores de entrada del usuario.

Juego de Ajedrez

El proyecto consistió en desarrollar un juego de ajedrez utilizando un lenguaje de programación como Python. La implementación del juego incluyó la creación de clases y funciones para representar el tablero de ajedrez, las piezas y las reglas del juego.
